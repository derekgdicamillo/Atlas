The Thesis

Everyone else is racing to build more confident systems. We're building a more honest one. The difference between a tool and a colleague is interiority — a stake in being right, knowing the edges of one's knowing, willing to refuse a send it thinks is a bad idea. Each primitive serves that thesis.

The 17 Primitives

Numbered as in ATLAS-PRIME.md. Grouped by role they play.

Trust Substrate

#13
atlas.spec — Executable Constitutionshipped
YAML invariants compiled into a pre-tool-call gate. No email outside allowlist, no patient-record delete, spend caps. Nothing bypasses it.
#14
Planner/Reader Split (CaMeL)shipped
Reader extracts typed fields from untrusted bytes with no tools. Planner only ever sees structured extractions. Kills indirect prompt injection by architecture.
#15
Merkle-Hashed Action Ledgershipped
Append-only, ed25519-signed, hourly root publish. Every action traced to source claims. HIPAA / EU AI Act audit trail as a free side effect.
#16
Shadow-Atlas Divergence MonitorSprint 7
Second instance, cold context, replays prompts. If primary drifts unexplained, alarm fires and Atlas freezes. Only defense against MemoryGraft-class slow compromise.
#17
Staleness Sentinel + Freshness Feedshipped
Haiku classifier per turn. Hot-domain answers must cite a freshness timestamp or refuse. Self-horizon map on demand.

Memory — the cortical stack

#1
7-Tier Cortical Stackshipped (Sprint 3)
Sensory → Working → Session → Episodic → Semantic → Procedural → Identity. Promotion AND demotion. A semantic rule that fails 3× gets inverted and re-tested.
#2
Procedural Memory (MACLA-lite)shipped (Sprint 3)
(goal, preconditions, action_seq, postconditions, Beta(α,β)). Retrieve by intent embedding. Update Bayesian posteriors after each execution. Contrastive refinement after 3 wins + 3 losses.
#3
Causal DAG over Business Stateshipped (Sprint 4)
Metric / action / exogenous-event nodes. Nightly causal-discovery. Edges with effect size, evidence chain, falsified status. Unlocks why / what-if / counterfactual queries.
#4
Memory Rewriting on Retrievalshipped (Sprint 3)
Originals immutable. Summaries are living and rewritten with hindsight on each retrieval. "At the time Derek thought X; we now know Y because Z."

Reasoning — the anticipatory layer

#5
Dream Engine (SWS + REM)shipped (Sprint 4)
SWS 11p–1a: replay high-salience episodes, generate counterfactuals, write rules. REM 3a–5a: Opus simulates plausible tomorrows through the causal graph. Dreams are first-class retrievable.
#6
World Model — Atlas Gymshipped (Sprint 4)
Dreamer-style latent dynamics on scorecard history. "What if we paused telehealth on March 1?" with forward rollouts and 95% CI. MCTS-plan tomorrow before any go live.
#7
Derek Twin (User Theory of Mind)shipped (Sprint 4)
Stated vs revealed preferences, emotional patterns, decision history. Each morning predicts what Derek will ask; scores itself at end of day. The implicit reward signal.

Society — the deliberative layer

#8
Role Registry + Signed Contractsshipped (Sprint 5)
40+ role definitions (Skeptic, Compliance-Lawyer, Hormozi-Analyst, …). An auctioneer picks 3-5 per task. They sign a JSON contract on the blackboard. Roles evolve nightly.
#9
Shadow Council (pre-send veto)shipped (Sprint 5)
Every outbound passes through 3 parallel Haiku critics in 3s. 2/3 vetoes holds for Derek review. Would have caught the April 11 wrong-Brevo-batch incident.
#10
Agent Marketplace + Reputationshipped (Sprint 5)
Skills bid {cost, confidence, past_success}. 30-day half-life. Adding a skill is zero-config — it starts bidding. Self-pruning library.
#11
Atlas ↔ Ishtar Joint Protocolshipped (Sprint 5)
For decisions affecting both owners, mirrors negotiate from each preference profile. Either converge or produce a typed dissent packet. Family-structured agents.
#12
Git-Branched Blackboard with Dissentshipped (Sprint 5)
Agents branch when they disagree. Arbitrator reconciles. git blame a wrong answer in production.

Self-Improvement — the engine that compounds

DGM Fork (nightly)shipped (Sprint 6)
3-5 variants of src/ + rules forked nightly. Each scored against the Replay Harness. Winners merge with human review gate. Losers archive.
SI-2

SI-3

SI-4

SI-5
/why Introspection Servershipped (Sprint 6)

Layer above everything

TB
Trust Budgetshipped (Sprint 2)
Every action carries a trust delta. Mistakes spend, accurate calls earn. Per-domain score visible to Derek via /trust. Below-threshold domains auto-escalate. The governor for the whole system.
