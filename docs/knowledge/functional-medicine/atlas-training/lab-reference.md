# Functional Medicine — Optimal Lab Reference (Cheat Sheet)

One-screen quick reference of the functional/optimal targets taught in the Spring 2026 mentorship. **This is the condensed lookup; full detail, sources, and evidence labels are in [`01-lab-optimization/optimal-vs-normal-ranges.md`](../01-lab-optimization/optimal-vs-normal-ranges.md).** Unless a study is named, treat targets as functional-medicine practitioner consensus, not guideline evidence.

## Glucose / insulin (root-cause panel — read first)

| Lab | Optimal target | Notes |
|---|---|---|
| Fasting insulin | **≤ 5 mcU/mL** | Most important early marker; can be high with "normal" glucose (Kraft patterns). Interpret alongside A1c, trigs, hormones. |
| HbA1c | **< 5.3%** | Course teaches complication risk begins ~5.3% (Butalia 2024 cohort). |
| Fasting glucose | ~80s mg/dL (~85) | Low-carb/keto can transiently raise fasting glucose without pathology. |
| Triglycerides | **< 90 mg/dL** | High trigs ≈ insulin resistance. |
| HDL | **> 50 mg/dL** | |

## Iron / ferritin

| Lab | Optimal target | Notes |
|---|---|---|
| Ferritin | **80–100** (card) / 80–120 (slide) ng/mL | Symptom floor ~65; **cardiac risk < 20**; **> 45 required before starting T3 meds**. Ferritin 1.5–2× iron ⇒ inflammation, not overload. |
| Iron | 80–120 (close to ferritin) | |
| Iron saturation | ~40% | |

## Thyroid (full panel — never TSH alone)

| Lab | Optimal target | Notes |
|---|---|---|
| Free T3 | **3.5–4 ng/dL** | The "gas pedal" — only thyroid molecule entering the cell for ATP. |
| Reverse T3 | **10–12 ng/dL** | The "brake pedal"; > 15 unmedicated / > 18–19 ⇒ inflammation. |
| TSH | — | Suppressed & uninformative once on liothyronine. |
| Free T4 | — | Neither TSH nor FT4 enters the cell. |
| TPO / TgAb | **< 100** | A single positive antibody = Hashimoto's; hundreds–thousands confirms. |

## Inflammation

| Lab | Optimal target | Notes |
|---|---|---|
| hs-CRP | **< 1 mg/L** | 3–10 = strong CV risk (Ridker/JUPITER); >10 ⇒ look beyond CVD. |
| WBC | **6.5–7.0 or less** | |
| Homocysteine | **5–8 (target ~7)**; flag > 8–9 | Marker of inflammation + MTHFR. High Hcy + normal MMA ⇒ folate; + high MMA ⇒ B12. |
| Uric acid | **< 5 mg/dL** | Normal-range uric acid still predicts HTN (Leiba 2015). |

## Liver / protein / electrolytes

| Lab | Optimal target | Notes |
|---|---|---|
| AST / ALT | ~20 IU/L | **Low** ⇒ malnutrition (protein/D/iron/iodine); high ⇒ liver stress/NAFLD. |
| Alk phos | ~60 IU/L | |
| Total protein | 7–7.4 g/dL | |
| Albumin | 4.1–4.4 g/dL | |
| Globulin | ~3.0 (2.9–3.2) g/dL | Building-block protein; low ⇒ poor intake/absorption. |
| Sodium | > 140 mEq/L | |
| Potassium | > 4.0 mEq/L | |
| Mg / Cr / Se | test as **RBC**, not plasma | |

## Hormones (mid-range unless noted; insulin is upstream)

| Lab | Optimal target |
|---|---|
| DHEA-S | mid-range (age/sex dependent) |
| Total testosterone | mid-range or higher |
| Free testosterone | upper 1/3 |
| Estradiol (E2) | closer to 100 pg/mL |
| Estrone (E1) | as low as possible; E1 should be < E2 |

## Vitamin D

| Lab | Optimal target | Notes |
|---|---|---|
| 25-OH vitamin D | **floor 60**, ceiling 80–100 ng/mL (sources vary) | Toxicity 200. Cofactors required: **magnesium, K2, boron, dietary fat**. |

## Pre-draw prep (from 01-lab-optimization)

Hold **biotin/B12 5–7 days** (≈1 day per 100 mcg), **10–12 h fasting**, **thyroid meds 18–20 h**, **iron 4 days**, other **hormones 2–3 days**, **vitamin D 10–14 days** only when confirming a true-high.
